Open source projects

All open source projects listed below are available on Github

Sneak

A version of the famous snake game that we used to love and play on our Nokia phones, but this time in your terminal. Download the project from PyPi.

Visit project on PyPi

MiniMe

MiniMe is a link shortening web app similar to Bitly. It utilises a Django Rest Framework backend with Python 3, Redis and Docker.

Frontend is built with Bootstrap, Vue.js and Less css. The project is published on Heroku infrastructure.

Visit minime.bluecap.se

Portfolio

This website. Running a Python Flask server in Docker. It is statically published on Github Pages.

Visit portfolio.bluecap.se

Yarr!

Yarr! is a small project that adds (another) 'I am lucky'-feature to The Pirate Bay.

The client is done in Python and published on PyPi.
The backend is done in Node.js and is published to NPM.

Visit client project on PyPi
Visit backend project on NPM
